# 🗓️ Wall Calendar

An interactive wall calendar built with React + Vite. Inspired by a physical wall calendar aesthetic.

## ✨ Features

- 📅 Monthly calendar grid with Monday-first layout
- 🖼️ Dynamic hero image per month with custom photo support (URL, file upload, YouTube, video)
- 🌙 Moon phase display calculated with pure JS math
- 🇮🇳 Indian public holidays marked with colored dots
- 📍 Pin custom events to any date with color labels
- 📆 Date range selection with hover preview
- 📝 Notes panel with auto-save to localStorage
- 🌗 Dark / Light mode toggle
- 🔔 Browser notifications for upcoming pinned events
- 📱 Fully responsive — desktop side-by-side, mobile stacked
- 🎬 3D page-flip animation on month change

## 🚀 Getting Started

### Prerequisites
- Node.js 18+

### Installation

```bash
git clone https://github.com/Niraj0916/wall-calender.git
cd wall-calender
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## 🏗️ Built With
